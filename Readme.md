
## Run Locally

Clone the project

```bash
  git clone https://github.com/saugatrajbhandari/hamro-doctor.gitt
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Setup database

```bash
  python manage.py migrate
```
```bash
  python manage.py makemigrations
```

Start the server

```bash
  python manage.py runserver
```
